Typography

Capgemini branding standards use Ubuntu Mono for primary headlines. Bold and Regular weights are both valid options for any headline. Ubuntu is Capgemini's font for all other headlines and for body text. Light, Regular, Medium, and Bold weights are valid options for body text. Don't use Light for small text.
